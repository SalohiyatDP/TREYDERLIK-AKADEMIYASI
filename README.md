# 🎓 TREYDERLIK AKADEMIYASI

Professional Offline Trading Education Platform

## 🌟 Xususiyatlar

- ✅ **100% OFFLINE** - Internet talab qilinmaydi
- 🔐 **No Login Required** - Registratsiya/login yo'q
- 💾 **Local Storage** - Barcha ma'lumotlar IndexedDB da saqlanadi
- 🇺🇿 **O'zbek tilida** - To'liq o'zbek tilida
- 🎨 **Dark/Light Mode** - Ikki xil tema
- 📱 **PWA** - Progressive Web App
- 🎯 **8 ta Modul** - Treyding asoslaridan ICT ga qadar
- 📊 **Interactive Charts** - TradingView Lightweight Charts
- 🎮 **Simulators** - Candle, Chart, Backtest simulyatorlari
- 📝 **Trading Journal** - Treyd jurnali va statistika
- 🎓 **Certification** - 100% tugallash uchun PDF sertifikat

## 🚀 Modullar

### 1️⃣ Treyding Asoslari (16 mavzu)
- Treyding, Broker, Forex, CFD, Crypto
- Volatility, Buy/Sell, Spread, Pip, Lot
- Margin, Leverage, Equity, Balance, Drawdown

### 2️⃣ Grafiklar
- Line Chart & Candlestick Chart
- Interactive Candle Simulator
- Drawing tools

### 3️⃣ Price Action
- Trend, Range, Breakout, Pullback, Reversal

### 4️⃣ Likvidlik
- Buy/Sell Side Liquidity
- Equal High/Low, Stop Hunt, Liquidity Grab

### 5️⃣ Smart Money Concepts (SMC)
- BOS, CHOCH, MSS
- Internal/External Structure

### 6️⃣ Order Block
- Bullish/Bearish OB
- Mitigation

### 7️⃣ Fair Value Gap (FVG)
- FVG, IFVG, BISI, SIBI

### 8️⃣ ICT
- PD Arrays, OTE, Liquidity Sweep
- Judas Swing, Silver Bullet
- Kill Zones, Daily/Weekly Bias

## 🛠️ Texnologiyalar

- **Frontend:** React 18 + TypeScript
- **Styling:** TailwindCSS
- **Animation:** Framer Motion
- **Charts:** Lightweight Charts (TradingView)
- **Routing:** React Router v6
- **State:** Zustand
- **Database:** Dexie.js (IndexedDB)
- **PWA:** Vite PWA Plugin
- **PDF:** jsPDF + html2canvas

## 📦 O'rnatish

```bash
npm install
```

> **Eslatma:** Birinchi marta o'rnatishda barcha dependencies yuklanadi. Bu bir necha daqiqa vaqt olishi mumkin.

## 🚀 Ishga tushirish

Development mode:
```bash
npm run dev
```

Browser avtomatik ochiladi va `http://localhost:5173` da ilova ishga tushadi.

## 🏗️ Build

Production build:
```bash
npm run build
```

Build fayllar `dist/` papkasida saqlanadi.

## 👀 Preview

Build qilingan ilovani ko'rish:
```bash
npm run preview
```

## 📱 PWA Install

Build qilgandan keyin `dist` papkasini web serverga joylashtiring va browserda oching. "Install" tugmasini bosing.

## 📂 Papka Strukturasi

```
src/
├── components/         # Reusable komponentlar
├── modules/           # 8 ta modul
├── pages/             # Page komponentlar
├── lib/               # Database va utilities
├── hooks/             # Custom hooks
├── store/             # Zustand stores
├── types/             # TypeScript types
└── styles/            # Global styles
```
